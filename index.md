# 스터디 개요

## 목적

* ORM library인 Ef-core를 연습한다.

* Domain-Driven Design를 기반으로 한 Reactive Architecture를 이용해 간단한 Backend 서비스를 만든다.

* rxjs, ngrx를 기반으로 한 Reactive Structure의 Frontend 서비스를 만든다.

* aws에 backend, frontend 서비스를 올린다.
